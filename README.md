# Gym
A gym website with small backend, when you submit form your data is save in output.txt


# Tech used 

**Frontend-** Pug, CSS

**Backend-** Node, Express

## Run Locally

Clone the project

```bash
  git clone https://link-to-project
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  nodemon app
```
